WIRECARD PAYMENT FOR DRUPAL COMMERCE
---------------

Payment method supported: WPP (Wirecard Payment Page v2) seamless mode.


INSTALLING WIRECARD MODULE
-------------------------------------

1. Download latest module from https://www.drupal.org/project/wirecard

2. Install Guzzle PHP HTTP client: https://github.com/guzzle/guzzle

3. Enable Wirecard as usual: /admin/modules


CONFIGURING PAYMENT METHOD
--------------------------

1. Create an account at https://www.wirecard.com

2. Insert your credentials at the Wirecard configuration page
   admin/commerce/config/payment-methods/manage/commerce_payment_wirecard_wpp
   Remember to test the functionality with the test credentials before going
   live!
